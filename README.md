# 📚 AllStudy – Responsive Education Website

AllStudy is a modern and responsive educational multipage website built using **HTML, CSS, Bootstrap, and JavaScript**. The project demonstrates frontend development skills such as responsive layouts, structured UI components, and interactive elements.

This project was created as part of my **web development learning journey and portfolio** to showcase practical implementation of frontend technologies.

---

## 🚀 Features

* Fully **responsive design** for mobile, tablet, and desktop
* Modern and clean **user interface**
* Structured **services and gallery sections**
* Contact page with **interactive enquiry form**
* Smooth navigation and organized layout
* Integration-ready structure for future backend/API features

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and layout
* **Bootstrap** – Responsive design framework
* **JavaScript** – Interactivity and form handling
* **Font Awesome** – Icons
